icon in circle
Chain free 3 / 24 Detached house inactive active / open dot

Circles (radius-full, 46px, surface fill + border) contain single arrows, icons or counts. Pills (radius-full) carry status / availability (accent), photo counts (dark) and category labels (surface-alt). Dots (9px accent — hollow when inactive, filled when active) mark slider pagination and expandable accordions. Directional arrows are always ← / → for carousels & sliders, and → for inline text links — never ‹ › or chevrons.

05 · Mobile is a separate pass
[ portrait 9:16 video / poster ]

We know your market best

Every page gets a dedicated mobile pass at ~390px — not just reflow. Art-directed media is handled inside each section: the video hero swaps to a portrait 9:16 crop with poster fallback; nav collapses to a hamburger with a sticky CTA; topbar utilities move into the menu; footer columns become accordions; property search toggles between full-screen map and list.

Touch targets stay ≥44px and the primary CTA stays thumb-reachable. These treatments will be applied per section as each page is built.

06 · Layout & responsive utilities
Section rhythm

Each band is var(--space-24) top & bottom. Where two same-background sections stack, the gap doubles — add et-tight-top / et-tight-bottom (or lower the inline padding to ~space-10 / space-6) so the seam reads as one rhythm, not a void.

Three-up → mobile carousel

et-cards3 renders an equal-height 3-column grid on desktop and a swipe carousel on mobile. Pair it with the et-feat-head controls: a text link on desktop, arrows + a “view all” link on mobile. Used by video reviews, testimonials and news.

07 · Interaction conventions
No toggle arrows on expandable controls

We do not add a chevron / toggle arrow to controls that expand a panel (e.g. the AI search assistant pill on the listings filter bar). Clicking the control itself reveals the panel — the arrow is redundant and removed everywhere. This applies to any expand-in-place pattern (what we call the “R4 Expanded” treatment).

Label the action, not the affordance

Lead an expandable trigger with a meaningful icon (e.g. a sparkle/AI mark for the AI search assistant) and a clear text label. Avoid generic disclosure widgets that only signal “there is more” without saying what.
